Project One

This project is a UCLA dining hall tracker (NOM-style clone). It fetches live menu data, organizes it by dining hall and meal, and allows filtering by dietary preferences.

Features

  • Live dining menu data
  • Meal filtering (breakfast/lunch/dinner)
  • Dietary tags (vegan, vegetarian)
  • Fast, minimal UI

Tech Stack

HTML, CSS, JavaScript (fetch API), optional Node backend
